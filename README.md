# Mapa-cinemas-Brasil-Power-BI

Para construir esse dashboard utilizei os dados disponíveis na base de dados da ANCINE. A base de dados tratada está disponível para download. Importante dizer que considerei apenas as salas de cinema em funcionamento. 

Este material foi elaborado como complemento de três episódios do podcast Ponto Jornalismo de Dados sobre o cinema. 
1. Filmes para zero pessoas: esse é o público mais recorrente nas salas de cinema em PG. [Ouça aqui](https://open.spotify.com/episode/7aMVJa8D2aNiHKHSwaqOpy?si=435a487692fe4355) 
2. O gosto do público ponta-grossense nas telas: sucessos e fracassos de bilheteria. [Ouça aqui](https://open.spotify.com/episode/4unKcxiyDcz0Zh9QnmjEzn?si=c6e5736b2d6e4cfe)
3. O cinema em PG: da ascensão às salas vazias. Entrevista com o professor Nelson Silva Junior. [Ouça aqui](https://open.spotify.com/episode/2hJAokcri1jBNf8H9BznsD?si=01a2d2645efb4f9d)
